# G-Score

Web application for looking up Vietnamese national high school exam scores. Search student results by registration number, view top-performing students in Group A (Math, Physics, Chemistry), and explore score distribution statistics across all subjects.

## Tech Stack

React 19, TypeScript, TanStack Router, TanStack Query, Tailwind CSS 4, Shadcn UI, Recharts, Vite

## Prerequisites

- Node.js >= 18
- npm

## Getting Started

```bash
# Install dependencies
npm install

# Start development server (http://localhost:5173)
npm run dev
```

## Scripts

| Command | Description |
| --- | --- |
| `npm run dev` | Start dev server on port 5173 |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run test` | Run tests (Vitest) |
| `npm run lint` | Lint with ESLint |
| `npm run format` | Format with Prettier |
| `npm run check` | Fix formatting + linting |
